# music_genre_classify_site
음악 데이터를 통해 장르를 분류하는 프로젝트 입니다.<br/>
분류 소프트웨어와 서버를 분리해서 설계하였습니다.<br/>
분류 결과를 Django를 분류 소프트웨어와 통신 가능하게 설계하였습니다.<br/>
Docker 와 volume 을 활용해 음악파일에 접근할 수 있게 설계하였습니다.(S3, NAS로 교체 가능하게 설계)<br/>
